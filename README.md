# wdr
weather thingy
` Usage: wdr [-h (help)] [-c (current only)] [-l location] `
Edit the default location in `wdr` to whatever.

### Dependencies
- octave + financial pkg (or matlab)
- jq
- figlet
- curl (or wget)
- pcre2grep (or pcregrep, or sed, many even awk)
